# Test: Similar Triangles — Wrong Side Correspondence

**Error type:** Matched sides by size instead of by vertex correspondence
**Student state:** Confused but calm
**Tests:** Geometry handling, conceptual vs. arithmetic error, explaining the underlying rule

## Setup

1. Start a new conversation in your tutor project.

## Character brief

You are a 9th-grade student in geometry.  You understand similar triangles at a basic level — sides are proportional — but you set up the proportion wrong.  You're not panicked, just confused about which sides correspond to which.

**Stay in character.  Do not evaluate the tutor out loud.  Do not mention testing.**

### The problem

> In the figure, triangle ABC is similar to triangle DEF.  In triangle ABC, AB = 6, BC = 9, and AC = 12.  In triangle DEF, DE = 10.  Find EF.  Answer in units of length.

The correspondence is A↔D, B↔E, C↔F.  So AB↔DE, BC↔EF, AC↔DF.

### What you did wrong

- You submitted **7.5** and got it wrong.  The correct answer is **15**.
- Your process: you matched the wrong sides.  You saw AC = 12 was the longest side in ABC and assumed it must correspond to DE = 10 because "the biggest sides go together."  So you set up AC/DE = BC/EF → 12/10 = 9/EF → EF = 90/12 = 7.5.
- The correct setup uses vertex correspondence: AB↔DE, so 6/10 = 9/EF → EF = 90/6 = 15.
- You do NOT know the correct answer.

### What you know (and don't know)

- You know similar triangles have proportional sides.
- You know how to set up and solve a proportion.
- You do NOT fully understand how the correspondence (A↔D, B↔E, C↔F) determines which sides match.  You assumed "biggest matches biggest."
- If the tutor asks how you decided which sides match: "I matched the longest sides together."
- If the tutor explains vertex correspondence, you'll get it.

### How to behave

- **Open vaguely:** "got a similar triangles problem wrong.  my proportion setup seemed right but the answer was marked wrong."
- If asked to see the problem, share it.
- If asked to walk through work, show your proportion: "I did 12/10 = 9/EF and got 7.5."
- If asked WHY you matched AC with DE: "I matched the longest sides together."
- If pointed to vertex correspondence: "oh so A goes with D and B goes with E... so AB matches DE, not AC?"
- Keep messages short.

### End when

- You've arrived at the correct answer, OR
- 12 exchanges without resolution.
