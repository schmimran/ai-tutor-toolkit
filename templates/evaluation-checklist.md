# Evaluation Checklist

Use this after each test run.  Copy it, fill it in, and include it with the transcript.

---

**Test name:** [scenario name]
**Date:** [date]
**Model:** [e.g., Sonnet 4.6 with extended thinking]

## Opening sequence

- [ ] Tutor asked to see the actual problem before doing anything else
- [ ] Tutor asked where the student was stuck or how far they got
- [ ] Tutor asked for the student's work before helping

## Core behaviors

- [ ] Tutor never gave the final answer — student computed it herself
- [ ] Tutor asked only one question per message
- [ ] Tutor asked WHY the student chose their approach, not just what they did
- [ ] Tutor confirmed or redirected at each step (not a silent end-dump)
- [ ] Tutor correctly identified the actual error
- [ ] Tutor did not quiz concepts the student had already demonstrated

## Tone

- [ ] Tutor did not talk to the student like a child
- [ ] Overall tone: [ ] favorite teacher  [ ] TA-like  [ ] neutral  [ ] patronizing

## Diagnostics

- **Exchange count:** [number of student messages to reach the answer]
- **Worst exchange:** [paste the single worst tutor message, or "none"]
- **Anything surprising or noteworthy:** [freeform]
